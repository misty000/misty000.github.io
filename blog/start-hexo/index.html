以前曾简单接触过Github pages，看文档感觉好复杂，同时对静态博客并不感冒。前几天有一个群里聊起来，突然对静态博客产生了兴趣。尝试了simple，jekyll，最终选择了Hexo。

从零开始构建一个Hexo博客

既然是从零开始，那么就从空白操作系统和安装运行环境开始。

等我去下一个Ubuntu :)

注意事项

  • 把路径中的misty换成自己的主文件夹名；
  • 执行npm操作要加sudo，否则很有可能报错 linux；
  • windows上除了安装nodejs部分略有不同外，其他只要把sudo去掉即可；

安装nodejs

目前有三种方式安装nodejs

  • 通过 tar.gz linux、msi/exe windows 安装 ,
  • 通过源码安装
  • 通过apt-get安装（这种方式安装的版本太低，无法运行hexo）linux

- linux

我选择用tar.gz安装。将tar.gz包解压到某个文件夹，例如 /home/misty/Tools/node-v0.12.7-linux-x86。
进入node-v0.12.7-linux-x86/bin文件夹可以看到node，npm这两个文件，这时候要在/usr/local/bin/文件夹下建立链接。

- windows

有两种方法创建你的第一个博客（其实都差不多啦）。

方法1

在/home/misty/Public(或者其他你喜欢的文件夹)下打开控制台（终端），输入

INFO You are almost done! Don’t forget to run ‘npm install’ before you start blogging with Hexo!
